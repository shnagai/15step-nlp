15Stepで踏破 自然言語処理アプリケーション開発入門
サンプルデータ・コード
==========================================

以下のフォルダが含まれます。

`dialogue_agent_data`
--------------------
対話エージェントを作成する例題で使用する学習データ、テストデータです。

`docker`
--------
執筆環境をDockerで再現するための`Dockerfile`と、ビルド時に使用する`requirements.txt`です。
Dockerを使わず環境構築する場合でも、`requirements.txt`は利用できるでしょう。

`src`
-----
サンプルコードの抜粋です。
